# Sistema de Reservas para Barbería

## Descripción
Sistema web completo para gestionar reservas en tu barbería. Los clientes pueden agendar, reagendar y cancelar citas, y tú tienes un panel de administración para gestionar todo.

## Características Principales

### Para Clientes:
- Registro y login de usuarios
- Visualización de servicios disponibles
- Agendamiento de citas
- Reagendamiento y cancelación de reservas
- Historial de reservas

### Para el Administrador (Barbero):
- Panel de control con estadísticas
- Gestión completa de reservas
- Administración de servicios
- Configuración de horarios
- Gestión de clientes
- Configuración del sistema

## Instalación y Uso

### 1. Requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Editor de texto (VS Code recomendado)

### 2. Estructura de Archivos
